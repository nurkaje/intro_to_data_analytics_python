# Intro to Data Analytics with Python

This repository contains my exercises and lab assignments for the **Introduction to Data Analytics with Python** course.

The project includes multiple Python scripts focused on:
- data loading and cleaning
- exploratory data analysis
- grouping and aggregation with Pandas
- data visualization
- working with real datasets from different domains

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/nurkaje/intro_to_data_analytics_python.git
````

2. Open the project folder:

   ```bash
   cd intro_to_data_analytics_python
   ```

3. Install the required libraries:

   ```bash
   pip install pandas matplotlib seaborn folium plotly numpy pillow
   ```

4. Run any script from its folder, for example:
   ```bash
   python python_itds_lab5/pandas1.py
   ```

## Notes
* Most scripts expect the corresponding CSV dataset to remain in the same folder as the script.
* Some visualizations open in a browser window or generate interactive output files.
* The repository is organized by labs, so each folder represents a separate set of exercises or tasks.
